## Overview

This is a modern, high-performance portfolio website designed for a **Senior Cloud Engineer**. It features a "Dark SaaS" aesthetic with deep zinc backgrounds, glassmorphism, and smooth animations.

Built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/r3garciaz/portfolio-nextjs.git
    cd portfolio-nextjs
    ```

2.  **Install dependencies:**
    ```bash
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    yarn dev
    ```

4.  **Build for production:**
    ```bash
    yarn build
    ```

## Project Structure

-   `src/components/Modern/`: New component architecture.
    -   `Sections/`: Hero (Full-width), About, Skills (Bento), Experience (Pipeline), Projects, Contact.
    -   `Layout/`: Glassmorphic Header & Footer.
    -   `UI/`: Reusable Cards and Section wrappers.
-   `src/data/`: Centralized content management.
